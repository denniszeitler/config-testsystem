# Configure-Testsystem

PowerShell automation script to set up a fresh Windows Server environment.  
This includes installing necessary PowerShell modules, promoting the server to a Domain Controller, and creating basic Active Directory structure and test objects.

---

## 🧠 Idea & Purpose

This project was built to quickly spin up a local or virtual test environment with a working Active Directory setup.  
It helps automate repetitive steps when preparing lab systems. 

In particular, the goal is to test scenarios using PowerShell Universal in an on-premises domain environment, to develop and validate scripts for Active Directory, infrastructure automation, and system management.

---

## 🎯 Goals

- ✅ Install required PowerShell modules
- ✅ Promote the server to a Domain Controller (new forest)
- ✅ Create default OUs (e.g., Users, Computers, ServiceAccounts)
- ✅ Create test users and groups
- ✅ Install VS Code
- ✅ Install Powershell Universal

---

## ⚙️ Requirements

- Windows Server 2019/2022
- Run with administrative PowerShell
- Internet access for module installation
